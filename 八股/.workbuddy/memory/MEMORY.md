## 项目说明
本项目是文档场景，用户需要创建面向业务的分布式系统实战手册。

- `分布式系统业务实战手册/` 目录：面向业务的分布式系统实战手册（v8.0）
  - 主目录 19 个文件：
    - `00-总览.md`：导读、学习路径（8阶段）、核心概念索引
    - `01-Redis业务场景实战.md`：Redis 19 节（含7.0新特性+底层原理八股）
    - `02-MySQL业务场景实战.md`：MySQL 13 节（含InnoDB底层原理八股）
    - `03-数据一致性专题.md`：索引文件，指向子目录
    - `04-分布式系统核心设计模式.md`：16 节（含网关/RPC/安全/性能）
    - `05-中间件组合架构实战.md`：9 节（含数据迁移）
    - `06-高频业务场景面试题.md`：20 道场景题
    - `07-分布式定时任务与调度.md`：12 节
    - `08-附录.md`：术语表+Checklist+复盘+速记卡
    - `09-电商交易核心系统设计.md`：支付/订单/库存/商品/购物车
    - `10-营销与用户系统设计.md`：优惠券/营销/用户/通知
    - `11-内容与搜索系统设计.md`：评论UGC/搜索/文件
    - `12-风控与数据系统设计.md`：风控/报表/批量/异步任务
    - `13-存储引擎深度对比.md`：B+Tree vs LSM Tree
    - `14-单体与分布式架构选型.md`：单体vs分布式优缺点深度解析
    - `15-链路追踪深度实战.md`：Trace/Span/采样/OpenTelemetry
    - `16-监控告警体系建设.md`：四层监控/SLI-SLO/告警分级
    - `17-日志体系设计.md`：分级/结构化/采集/TraceID关联
    - `18-错误处理与异常体系.md`：错误分类/错误码/重试降级
    - `19-SRE与混沌工程.md`：SRE/混沌/全链路压测/故障应急

    - 00-导读 → 11-分布式时间与逻辑时钟

- 技术选型不单独成篇，应融入各业务场景讨论，体现思辨性
